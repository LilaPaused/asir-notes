# WAN y automatización

## WAN

Una WAN conecta redes separadas geográficamente.

Puede unir:

- Sedes.
- Centros de datos.
- Proveedores.
- Servicios cloud.
- Redes de clientes.

## ISP

Un ISP proporciona conectividad hacia otras redes.

En laboratorios, el ISP normalmente:

- Enruta tráfico público.
- Conecta con servidores externos.
- Puede propagar una ruta por defecto.
- No suele hacer NAT por el cliente.

## Internet en laboratorios

La nube de Internet representa una red intermedia o proveedor.

Un servidor conectado a esa nube no es Internet; es un host alcanzable a través de la red del proveedor.

## Automatización

La automatización permite gestionar dispositivos y configuraciones de forma repetible.

Objetivos:

- Reducir errores manuales.
- Aplicar cambios de forma consistente.
- Documentar configuraciones.
- Repetir despliegues.
- Facilitar auditoría.

## Ejemplos de automatización

- Plantillas de configuración.
- Inventarios.
- Scripts de verificación.
- Copias automáticas de configuración.
- APIs.
- Herramientas de gestión.

## Relación con redes

En redes modernas, la administración no se limita a configurar manualmente por consola.

Cada vez es más habitual:

- Control centralizado.
- Automatización.
- Monitorización.
- Validación de estado.
