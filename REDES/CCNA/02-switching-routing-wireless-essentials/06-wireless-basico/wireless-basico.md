# Wireless básico

## Concepto

Wireless permite la conexión a la red sin cable físico, usando radiofrecuencia.

## Dispositivos

- Access Point.
- Cliente inalámbrico.
- Router inalámbrico.

## Estándares

| Estándar | Uso |
|---|---|
| 802.11 | WiFi |
| 802.15 | Bluetooth |
| 802.16 | WiMAX |

## SSID

Nombre de la red inalámbrica.

## BSSID

Identificador asociado al punto de acceso, normalmente relacionado con su MAC.

## Seguridad inalámbrica

Conceptos habituales:

- Autenticación.
- Cifrado.
- Contraseña segura.
- Segmentación mediante VLANs.
- Red de invitados separada.

## Problemas habituales

- Interferencias.
- Baja señal.
- Canales saturados.
- Mala ubicación del AP.
- Seguridad débil.
