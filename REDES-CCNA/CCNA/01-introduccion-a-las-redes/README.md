# CCNA 1 - Introducción a las redes

Curso centrado en los fundamentos de redes: dispositivos, modelos de comunicación, direccionamiento IP, capa física, capa de enlace, Ethernet, ARP, ICMP, IPv4, IPv6, capa de transporte y comandos básicos de verificación.
