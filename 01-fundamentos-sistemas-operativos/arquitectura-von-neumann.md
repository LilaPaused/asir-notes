# Arquitectura de Von Neumann

## Qué es

La arquitectura de Von Neumann es un modelo clásico de organización de un ordenador.

Define que un sistema informático está formado por:

- CPU
- memoria principal
- entrada/salida
- buses de comunicación

## Esquema básico

CPU <-> Memoria principal <-> Entrada/Salida

Buses principales:

- bus de datos
- bus de direcciones
- bus de control

## CPU

La CPU es el componente encargado de interpretar y ejecutar instrucciones.

Partes importantes:

- unidad de control
- ALU
- registros

### Unidad de control

Coordina las operaciones internas del procesador.

Indica qué instrucción se ejecuta y cuándo.

### ALU

La ALU realiza operaciones aritméticas y lógicas.

Ejemplos:

- sumas
- restas
- comparaciones
- operaciones booleanas

### Registros

Son pequeñas memorias internas muy rápidas.

Guardan datos temporales que la CPU necesita usar inmediatamente.

## Memoria principal

La memoria principal almacena temporalmente:

- instrucciones
- procesos
- datos en uso

Normalmente se asocia con la memoria RAM.

Es volátil, por lo que al apagar el equipo se pierde su contenido.

## Entrada/Salida

La entrada/salida permite comunicar el sistema con el exterior.

Ejemplos:

- teclado
- ratón
- pantalla
- impresora
- disco duro
- tarjeta de red
- USB

## Buses

Los buses conectan los componentes del sistema.

### Bus de datos

Transporta información entre CPU, memoria y dispositivos.

### Bus de direcciones

Indica en qué posición de memoria se debe leer o escribir.

### Bus de control

Transporta señales de control y sincronización.

## Resumen rápido

| Componente | Función |
|---|---|
| CPU | Ejecuta instrucciones |
| Memoria principal | Guarda datos e instrucciones temporalmente |
| Entrada/Salida | Comunica el sistema con periféricos |
| Bus de datos | Transporta datos |
| Bus de direcciones | Indica posiciones de memoria |
| Bus de control | Coordina operaciones |

## Para examen

La arquitectura de Von Neumann se basa en una CPU conectada a la memoria principal y a los dispositivos de entrada/salida mediante buses de datos, direcciones y control.
