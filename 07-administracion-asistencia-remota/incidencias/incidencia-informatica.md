# Incidencia informática

## Qué es

Una incidencia informática es cualquier reporte de un usuario sobre un problema técnico que afecta a su trabajo.

Ejemplos:

- una aplicación no abre
- un equipo va lento
- no hay conexión a red
- teclado o ratón roto
- pantalla sin señal
- impresora no imprime
- pérdida de datos
- error de inicio de sesión

## Diferencia entre incidencia y petición

| Tipo | Ejemplo |
|---|---|
| Incidencia | Algo que funcionaba ha dejado de funcionar |
| Petición | El usuario solicita algo nuevo |

Ejemplo:

- Incidencia: “No puedo acceder a mi carpeta compartida”.
- Petición: “Necesito acceso a una carpeta nueva”.

## Datos mínimos que debe aportar el usuario

- ID de empleado.
- Correo o teléfono de contacto.
- Dispositivo afectado.
- Descripción del problema.
- Si hay datos importantes que recuperar.

## Datos que debe rellenar el técnico

Al recibir la incidencia:

- fecha de recepción
- técnico asignado
- categoría
- equipo afectado
- diagnóstico inicial
- prioridad real
- si se puede resolver remotamente

Al resolverla:

- fecha de resolución
- solución aplicada
- si hubo sustitución de equipo
- si se recuperaron datos
- piezas usadas
- si se reclamó garantía
- observaciones finales

## Objetivo

Una buena incidencia debe permitir que otro técnico entienda:

1. qué pasaba
2. qué se hizo
3. qué queda pendiente
4. cómo evitar que se repita
