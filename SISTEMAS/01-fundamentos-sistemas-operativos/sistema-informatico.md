# Sistema informático

## Qué es

Un sistema informático es un conjunto de elementos que permite recibir datos, procesarlos, almacenarlos y generar una salida útil.

Flujo básico:

Entrada de datos -> Proceso -> Salida de datos
                         |
                  Almacenamiento

## Componentes principales

Un sistema informático se compone de tres partes principales:

1. Hardware
2. Software
3. Componente humano

## Hardware

El hardware es la parte física del sistema.

Ejemplos:

- CPU
- memoria RAM
- placa base
- disco duro o SSD
- tarjeta de red
- teclado
- ratón
- monitor
- cableado
- periféricos

## Software

El software es la parte lógica del sistema.

Ejemplos:

- sistema operativo
- drivers
- aplicaciones
- herramientas de diagnóstico
- programas de usuario
- compiladores
- scripts

## Componente humano

El componente humano son las personas que interactúan con el sistema.

Puede incluir:

- usuario final
- técnico de sistemas
- administrador de red
- programador
- operador
- responsable de soporte

## Ejemplo práctico

Cuando un usuario abre un navegador:

1. El usuario realiza una acción.
2. El sistema operativo recibe la petición.
3. El hardware ejecuta instrucciones.
4. La red permite acceder a Internet.
5. El navegador muestra la web.

## Resumen rápido

| Elemento | Función |
|---|---|
| Hardware | Parte física |
| Software | Parte lógica |
| Usuario | Interacción con el sistema |
| Datos | Información procesada |
| Sistema operativo | Intermediario entre usuario, software y hardware |
