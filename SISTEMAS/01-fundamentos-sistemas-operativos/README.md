## Ideas clave

Un sistema operativo es el software base que permite usar el hardware de una máquina.

Entre sus funciones principales están:

- gestionar procesos
- gestionar memoria
- controlar dispositivos
- administrar archivos
- gestionar usuarios
- participar en el arranque del sistema

Este tema también incluye una parte práctica importante: creación de máquinas virtuales, instalación de sistemas operativos, configuración de arranque dual, reparación de GRUB y uso de Ventoy.
