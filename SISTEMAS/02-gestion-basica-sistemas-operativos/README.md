# Gestión básica de Sistemas Operativos

Este bloque recoge apuntes prácticos sobre administración básica de sistemas Windows y Linux.

La organización está hecha por conceptos técnicos, no por número de práctica, para que sea más rápido encontrar lo que se necesita.

## Contenido

### Terminales

- CMD
- PowerShell
- Bash
- Rutas, comodines y redirecciones

### Ficheros y directorios

- Crear, copiar, mover, renombrar y eliminar directorios
- Crear, mostrar, modificar y eliminar ficheros
- Árboles de directorios
- Backups y restauraciones
- Enlaces simbólicos

### Discos y particiones

- Administración de discos en Windows
- Diskpart
- PowerShell para discos
- GParted
- fdisk
- mkfs.ext4
- etiquetas de particiones

### Red y DNS

- Red interna en VirtualBox
- IP estática
- Firewall e ICMP
- DNS en Windows Server
- DNS en Ubuntu Server con Bind9
- nslookup y dig

### Usuarios y grupos

- Usuarios locales
- Grupos locales
- Políticas de contraseña
- Bloqueo de cuentas

### Exámenes

- Repaso PF2

## Idea principal

Este tema es la base práctica de administración: moverse por terminal, manipular archivos, preparar discos, configurar red, montar DNS y gestionar usuarios.
